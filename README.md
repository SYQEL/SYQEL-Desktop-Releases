# SYQEL Desktop Releases

Welcome to the official releases repository for **SYQEL Desktop** — a real-time music visualization engine built for **VJs, DJs, Creators and live performers**, while still simple & easy enough to use for immersive listeners.

---

## 🚀 What is SYQEL Desktop?

**SYQEL Desktop** turns your audio into **live, reactive visual experience** — instantly.

Designed for performance environments, SYQEL eliminates the friction of traditional VJ software and replaces it with a fast, modern, and flexible visual engine.

Whether you're on stage, streaming, or creating content, SYQEL gives you **visuals that move with your sound — in real time.**

---

## ⚡ Built for Performance

### 📴 Offline Mode
- No internet required  
- Reliable for live environments and travel  

---

### ❌ No Audio Driver Required!
Forget Routing Nightmares. No Loopback. Blackhole. VB Cable or Mixline required!

SYQEL works directly with your audio, removing your need for:
- Virtual audio cables  
- Loopback tools  
- Complex signal chains  

Just launch SYQEL and **click refresh** — to find your audio app.

## 🎛️ Core Features

### 🎵 Real-Time Audio Reactive Engine
- Instant response to system audio, decks, or mic input  
- Frequency-driven visuals (bass, mids, highs)  
- Built for low-latency performance  

---

### 🤖 AI-Powered Visual Generation
- Generate visuals from prompts  
- Create new looks instantly during sets  
- Endless variation without preloading assets  

---

### 🧠 Bring Your Own Visuals (or Build Them)
- Import **Milkdrop presets** (thousands available)  
- Build custom visuals with **ThreeJS**  
- Turn SYQEL into your personal VJ engine  

---

### 🎚️ Live Control Ready
- MIDI controller support  
- Keyboard shortcuts  
- Real-time parameter control during sets

### 📡 Stream & Stage Integration

- Perfect for clubs, stages, and online shows  

---

### 🍎🪟 Cross-Platform
- macOS + Windows support  
- Built for real-world performance rigs

### 🎨 Massive Visual Library

- Mix, match, and customize your style
